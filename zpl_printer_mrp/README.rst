This module connects the zpl_printer module with the mrp module and adds a printing option for
manufacturing orders to print the lot / serial for the producing units.

**Table of contents**

.. contents::
   :local:

Usage
=====

In the manufacturing order print serial numbers
with the option `Finished Product Label (Send to ZPL-Printer)`.
